# task-1: Project Scaffolding

## Details

Date: 2026-06-13

Dependencies: none

Decisions: none

## Description

Initialize the Node.js project with all tooling (package management, linting, formatting,
testing) configured and working. This is the foundation all other tasks depend on.

## Acceptance Criteria

- The system shall have a package.json with Node.js >=22 engines and jest/eslint/prettier as dev dependencies
- When `npm install` is run, all dependencies shall install without errors
- When `npm run lint` is run, ESLint shall execute with zero violations
- When `npm test` is run, Jest shall execute successfully with at least one passing test
- When `npm start` is run, the app shall start without crashing
- The .gitignore shall exclude node_modules and .env

## Implementation Plan

1. Create `package.json` with project metadata, scripts, engines, and dependencies
2. Create `.eslintrc.json` with ESLint + Prettier integration
3. Create `.prettierrc` with formatting rules
4. Create `.env` with placeholder environment variables
5. Create `.gitignore` for Node.js projects (including .env)
6. Create `src/index.js` - placeholder entry point
7. Create `src/test/index.test.js` - placeholder passing test
8. Run `npm install` to generate `package-lock.json`

## Test Strategy

- `npm install` - exit code 0
- `npm run lint` - exit code 0, no errors
- `npm test` - exit code 0, 1 test passing
- `npm start` - starts without error (kill after confirming startup log)
