# Task 26: Video-to-Fixture Matcher

## Details

Date: 2026-06-19

Dependencies: task-25

Decisions: none

## Description

Build an adapter module (`src/highlights/adapter.js`) that matches playlist
video items to local fixtures by parsing team names from video titles. The
adapter supports full country names, FIFA codes, and country aliases - all
case-insensitive and diacritic-stripped. Both home and away teams must appear
in the title for a match. Videos without "Highlight" in the title are skipped.

## Acceptance Criteria

- (Ubiquitous) The system shall export a `matchVideosToFixtures(videos,
  fixtures)` function that returns an array of
  `{ fixture, videoId, url, title }` objects
- (Event-driven) When a video title contains both home and away team names
  for a fixture, the system shall match that video to that fixture
- (Event-driven) When matching team names, the system shall normalize via NFD
  decomposition + strip combining marks + lowercase
- (Event-driven) When matching, the system shall check against `teams.json`
  full names, team IDs (FIFA codes), and `countries.json` aliases
- (Event-driven) When a video title does not contain "Highlight"
  (case-insensitive), the system shall skip it
- (Event-driven) When a video matches multiple fixtures, the system shall
  pick the fixture with the smallest `abs(videoPublishedAt - fixtureKickoff)`
- (Event-driven) When a fixture matches multiple videos, the system shall
  prefer the title containing "Highlights" over "Extended" or "Full Match";
  deterministic fallback: lower fixture ID
- (Ubiquitous) The system shall log unmatched videos for observability

## Implementation Plan

1. Create `src/highlights/adapter.js`
2. Implement `normalize(text)` helper:
   - NFD decomposition (`text.normalize('NFD')`)
   - Strip combining marks (replace `/[̀-ͯ]/g` with '')
   - Lowercase
3. Build a name lookup map at module load from `teams.json` and
   `countries.json`:
   - For each team: map normalized full name -> teamId (e.g., "mexico" -> "MEX")
   - For each team: map exact FIFA code lowercase -> teamId (e.g., "mex" -> "MEX").
     FIFA codes are short uppercase; store lowercase for lookup but do not
     apply full NFD normalization to them
   - For each country with aliases: map each normalized alias -> teamId
   - Both `teams.json` and `countries.json` share the same `id` field so no
     cross-reference conflict
4. Implement `extractTeams(title)`:
   - If title contains `|`, take portion before first `|` as matchup text.
     If no `|` present, use full title (FIFA format always has `|` but
     non-standard titles may not; still attempt match)
   - Split matchup text on ` v `, ` vs `, ` - ` (regex with surrounding spaces)
   - Each split result is a whole candidate (e.g., "Mexico", "South Africa");
     normalize and look up as a whole string in name map (not substring
     matching) to avoid false positives (e.g., "Iran" inside "Mirandola")
   - Return set of matched teamIds
5. Implement `matchVideosToFixtures(videos, fixtures)`:
   - Filter videos: skip if title does not contain "highlight" (lowercase
     check)
   - For each remaining video, call `extractTeams(title)`
   - For each fixture, check if both `homeTeamId` and `awayTeamId` are in
     the extracted teams set
   - Handle ties:
     a. Video matches multiple fixtures: pick closest by
        `abs(Date.parse(video.publishedAt) - Date.parse(fixture.dateAndTimeInUTC))`
     b. Fixture matches multiple videos: score by
        `title.toLowerCase().includes('highlights')` (plural, score 2) >
        `title.toLowerCase().includes('highlight')` (singular, score 1) >
        neither (score 0). Highest score wins. If tied: lower fixture ID
        (deterministic stable fallback)
   - Early return `[]` if videos or fixtures array is empty
   - Collect unmatched videos (matched zero fixtures), log them
   - Return array of `{ fixture, videoId, url, title }`
6. Create test file `src/test/highlights-adapter.test.js`

## Test Strategy

Run `npm test` and `npm run lint`.

- Test exact name match (e.g., "Mexico v South Africa")
- Test FIFA code match (e.g., "MEX v RSA")
- Test alias match (e.g., "South Korea v USA" matching KOR and USA)
- Test diacritics (e.g., "Turkiye" with and without special chars)
- Test multi-fixture day: two fixtures same day, video matched to correct one
- Test skip non-highlight: video titled "Press Conference" is excluded
- Test tie-breaking video->fixture: closest publishedAt wins
- Test tie-breaking fixture->video: "Highlights" preferred over "Extended"
- Test unmatched video: logs warning, not in result
- Test empty inputs: empty videos array, empty fixtures array
