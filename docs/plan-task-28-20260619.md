# Task 28: Manual Sync Command

## Details

Date: 2026-06-19

Dependencies: task-27

Decisions: none

## Description

Add `npm run sync:highlights` script for operator backfill. Fetches the full
highlight playlist (all pages, no date filter), runs the adapter against ALL
local fixtures, and enriches fixtures.json with `highlightsURL` for every
match found. No digest generation - data enrichment only.

Follows the existing pattern in `scripts/sync-data.js` (subcommand-based CLI
with env validation, rate logging, and graceful error handling).

## Acceptance Criteria

- (Ubiquitous) The system shall provide a `npm run sync:highlights` command
  that fetches the full playlist and backfills highlightsURL for all
  matchable fixtures
- (Event-driven) When the command runs, the system shall report: total
  videos fetched, matched count, unmatched count, fixtures updated
- (Event-driven) When HIGHLIGHTS_API_KEY is not set, the system shall exit
  with a clear error message (exit code 1)
- (Ubiquitous) The command shall not generate or post any digest (data only)
- (Ubiquitous) The command shall reuse the existing client, adapter, and
  enrich modules

## Implementation Plan

1. Create `scripts/sync-highlights.js`:
   a. Validate `HIGHLIGHTS_API_KEY` env var; exit 1 with message if missing
   b. Import: `fetchPlaylistItems` from `src/highlights/client.js`
   c. Import: `matchVideosToFixtures` from `src/highlights/adapter.js`
   d. Import: `enrichFixtures` from `src/highlights/enrich.js`
   e. Load `src/data/fixtures.json` (all fixtures, no date filter)
   f. Call `fetchPlaylistItems()` -> all playlist videos
   g. Call `matchVideosToFixtures(videos, fixtures)` -> matches
   h. Call `enrichFixtures(matches)` -> count of fixtures updated
   i. Log summary: total videos, matched, unmatched, updated
   j. Exit 0 on success
2. Add to `package.json` scripts:
   `"sync:highlights": "node --env-file=.env scripts/sync-highlights.js"`
3. Create test file `src/test/sync-highlights.test.js`

## Test Strategy

Run `npm test` and `npm run lint`.

- Test successful run: mocks client/adapter/enrich, verifies pipeline
  called in correct order with correct args
- Test missing HIGHLIGHTS_API_KEY: verifies exit with error message
- Test empty playlist (0 videos): logs summary, exits cleanly
- Test adapter returns 0 matches: logs summary, exits cleanly
